RxJS Username Beispiel

Registrierung

Username: Noch frei Bereits vergeben

Dieses Beispiel simuliert ein Nutzernamen-Eingabefeld eines Registrierungsformulars einer Webanwendung. Zur Registrierung darf der Nutzername im System noch nicht vergeben sein. Dazu wird während des Tippens ein REST-Webservice angerufen, um die Verfügbarkeit des aktuell eingegeben Wertes zu prüfen.

Zum Testen wurden im Server u.a. die am häufigsten in Deutschland vergebenen Vornamen bereits "registriert". Werden diese Namen eingetippt, erscheint eine Meldung, dass der Name bereits vergeben ist.

Einige bereits vergebene Nutzernamen zum Testen:

  • tester
  • testuser
  • sophie
  • marie
  • emma
  • anna
  • paul
  • alexander
  • ben
  • leon
  • lukas

Inspiriert wurde diese Beispiel durch den Vortrag "The enemy of the state" von Philip Roberts. Dieser Vortrag kann hier angeschaut werden: vimeo.com/68987289.

Die Anwendung in diesem Vortrag wurde jedoch mit Bacon.js implementiert, während das hier vorliegende Beispiel mit RxJS umgesetzt wurde.
